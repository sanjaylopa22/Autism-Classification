# Autism-Classification
Autism Classification using Deep Learning

This project predicts autism spectrum disorder using machine learning models.

## Features
- Data preprocessing
- Model training
- Evaluation metrics

## Models used
VGG16, ResNet-50, InceptionV3, EfficientNet-B0, and EfficientNet-B7 

## How to run

pip install -r requirements.txt

python src/Autism_Classification.ipynb
